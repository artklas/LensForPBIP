**LensForPBIP** is a professional documentation and analysis suite designed specifically for Microsoft Power BI Project (**PBIP**) files. It bridges the gap between raw metadata and actionable insights, allowing developers to audit data models, measures, and relationships with ease.

---

## üì• Download & Installation

To get started with LensForPBIP, follow these steps:

1. **Download:** Head over to the [**Latest Releases**](https://github.com/artklas/LensForPBIP/releases/latest) page.
2. **Installer:** Download `LensForPBIP_Setup.exe`.
3. **Install:** Run the setup and follow the on-screen instructions.
   
> [!TIP]
> **Windows SmartScreen Note:** Because this application is currently unsigned, Windows may display a "Windows protected your PC" message. Click **"More info"** and then **"Run anyway"** to proceed with the installation.

## ‚ú® Key Features

* **PBIP Integration:** Full support for the native Power BI Project format.
* **Measure Auditing:** Detailed breakdown of DAX expressions and dependencies.
* **Relationship Mapping:** Visualize and audit the links within your data model.
* **Report Structure Analysis:** Comprehensive insights into the report-layer metadata.
* **Standalone Execution:** No Python installation or environment setup required.

## üöÄ Quick Start

1. Open **LensForPBIP** from your Start Menu.
2. Browse to your local **.pbip** directory.
3. Select the analysis module you wish to run.
4. Export your documentation for sharing or auditing.
